# Hackerbay Backend Task

Stateless Node.js microservice with the following features:
- Authentication
- JSON patching
- Image Thumbnail Generation

# Docker Installation

`docker run -d -e PORT=9000 -e JWT_SECRET=secret -p 9000:9000 sirakoff/backend-task`

# Node Installation

`npm install`

`npm start`
